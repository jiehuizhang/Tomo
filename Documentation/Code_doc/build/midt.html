3.4.3. midt module¶

Multi Instance Desiicion Tree Classfication

midt.ClusteringtoBags(coord, dis_thresh=15)[source]¶

Clustering ROIs in slices into 3D ROIs, so that 2D ROI with close (x,y) coordination from adjacent slices will be assigned into one bag.

coord: list of coordination
The coordination list for all ROIs
dis_thresh:
Threshold for assigning ROIs into the same bag.

cancerList:

controlList:
List of negative data
