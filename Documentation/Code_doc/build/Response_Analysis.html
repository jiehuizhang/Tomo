3.1.2. Response_Analysis module¶

This file includes functions that analyzing the filtering responses. Specifically, creating batch response, taking the voting procedure,etc.

Response_Analysis.cerat_batch_response(response, sampRate, winSize)[source]¶

This function creats the batch response matrix in which each element is the mean value of a neighbour area from the corresponding response matrix

response: numpy array
Input response matrix
sampRate: integer
Sampling rate
winSize: integer
Size of the neighbourhood
Response_Analysis.dump_vector(response)[source]¶

This function reorgazies the responses derived from the same kernel set into an numpy array. (For debugging use only)

Response_Analysis.getCDF(temp_response)[source]¶

This function calculates voting score of response magnitude. The score is calculated as the rank of sorted magnitude.

temp_response: numpy array
Input batch response matrix
Response_Analysis.getODF(response, threshold=0.1)[source]¶

This function calculate the orientation voting score. The score in each orientation is set to 1 if it is above the percentage threshold, other wise set to zero

response: numpy array (2D)
Input batch response
threshold = 0.2
for num_orientation = 4
threshold = 0.1
for num_orientation = 8
Response_Analysis.integrating_poll(response, sampRate, winSize, shape)[source]¶

This function reverse the sampling proceduring after the voting result is derived

response: numpy_array
Input voting result
sampRate: integer
sampling rate
winSize: integer
neighbourhood size
Response_Analysis.vote(response, alpha=1.1)[source]¶

This function procedes the voting procedure. Magnitude score are calculated in function getCDF Orientation score are calculated in function getODF

response:
Input of batch response
alpha:
weight of orientation score alpha = 1.1 for num_orientation = 4 alpha = 0.55 for num_orientation = 8
