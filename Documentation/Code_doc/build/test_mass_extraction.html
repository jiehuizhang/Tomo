6.2. test_mass_extraction module¶

This script shows how to run suspicious mass extraction start from raw images. Preprocessing are included within the parallel process.

test_mass_extraction.test_func()[source]¶

Please specify the image tiff stack directory, file name and output path. Extracted masses list will be save in a workspace file for further use. Also please specify how many cores will be allocated for the parallel process.
