6.1. test_format_convert module¶

This script shows how to run the image format converting from smv to tiff(3D). The scipt only runs in linux system due to windows python buffering problem.

test_format_convert.test_func()[source]¶

You can conver a single smv file to 3D tiff by specifying data_path, output_path, fileName and set the SigleConvert flag to 1.

You can conver a batch of smv files to 3D tiffs by specifying root data_path, output_path and set the BatchConvert flag to 1.
